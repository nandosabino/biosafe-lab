# 🧪 BioSafe Lab

Um jogo interativo sobre **biossegurança em laboratório**, desenvolvido para simular tomadas de decisão em ambientes de risco controlado.

---

## 🎮 Sobre o projeto

O **BioSafe Lab** é um quiz gamificado que desafia o jogador a aplicar conceitos de biossegurança, avaliando:

* 🧠 Conhecimento teórico
* ⚡ Tempo de resposta
* 🎯 Tomada de decisão

O sistema adapta automaticamente a dificuldade das perguntas com base no desempenho do jogador, tornando a experiência dinâmica e envolvente.

---

## ✨ Funcionalidades

* 🎯 Perguntas com dificuldade adaptativa (fácil → médio → difícil)
* ⏱️ Sistema de tempo por pergunta
* 🧠 Pontuação baseada em dificuldade + tempo
* 🏆 Ranking com backend e banco de dados
* 🥇 Destaque dos melhores jogadores
* 🎮 Interface animada com Framer Motion
* 📱 Layout responsivo

## 🧠 Lógica do jogo

O sistema utiliza:

* Seleção dinâmica de perguntas
* Ajuste automático de dificuldade
* Cálculo de pontuação baseado em:

  * dificuldade
  * tempo restante
* Ranking persistente via API

## 🚀 Como rodar localmente

### 1. Clonar repositório

```bash
git clone https://github.com/seu-usuario/biosafe-lab.git
```

### 2. Frontend

```bash
cd frontend
npm install
npm run dev
```

### 3. Backend

```bash
cd backend
npm install
npm run dev
```
